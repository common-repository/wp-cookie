This is one of the easiest cookie notification plugin. It allows discreetly inform visitors that your website uses cookies.

== Description ==

It creates a small bar on top/bottom of a website with a short message that the website uses cookies with accept/decline button. Once a visitor accept/decline, then it disappears.

== Installation ==

1. Unzip the downloaded zip file and upload the plugin folder into the `wp-content/plugins/` directory. Alternatively, upload from Plugins >> Add New >> Upload Plugin.
2. Activate the plugin from Plugins page.

= Usage =

After successfully installing and activating the plugin, you will find “WP Cookie” submenu under "Settings" menu on left column of WordPress dashboard. Configure any options as desired using “General Settings” and "Style Settings" pages.
